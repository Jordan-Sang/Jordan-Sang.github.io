This project requires a backend server to function properly. To run the server, follow these steps:

  1. Download the server files by clicking here. Unzip the file once it finishes downloading
  2. Open the Terminal on your computer and navigate to the server folder that you just downloaded by using the cd command
  3. Once in the folder, run npm install to download the necessary resources to start the server. If you do not have Node.js installed on your computer, download it here
  4. Once the required packages have downloaded, run node server.js in your Terminal to start the server. You should see the following output:

    5. Server listening on port 4001
